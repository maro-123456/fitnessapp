FitLife Pro 💪🥗

FitLife Pro est une application web fullstack dédiée au suivi du fitness et de la nutrition.
Elle permet aux utilisateurs de gérer leur entraînement, suivre leur progression corporelle et accéder à des plans nutritionnels personnalisés dans une interface moderne et interactive.

Fonctionnalités principales:
-Authentification sécurisée avec JWT
-Bibliothèque d’exercices avec système de favoris
-Tableau de bord avec statistiques et suivi de progression
-Plans nutritionnels personnalisés avec export PDF
-Suivi quotidien de l’hydratation
-Carte interactive des salles de sport via Google Maps
-Citations motivationnelles via API

Technologies utilisées:

  Frontend
    * React
    * React Router
    * Axios
    * Recharts  
  
  Backend
    * Node.js
    * Express.js
    * MongoDB + Mongoose
    * JWT / bcrypt

APIs intégrées:
  * Quotes API (citations motivationnelles)
  * Google Maps API (localisation des salles de sport)
  * Nutrition Plan API (Génère des plans nutritionnels personnalisés)
  * Hydration Tracking API (suivre la consommation quotidienne d’eau et les objectifs d’hydratation)
